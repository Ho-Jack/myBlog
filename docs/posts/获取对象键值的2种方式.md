##  获取对象键值的方式

获取对象键值的方式

```js
var obj={
      a:1,
      b:2
}
 ① obj.a
 ② obj['a']

```

遍历对象 获取键值

```js
var obj={
      a:1,
      b:2
}
Object.values(obj)    // [1,2]
for(let key in obj){
    console.log( obj[key] ) //可以打印 遍历的 key带 ""     obj['a']
     console.log( obj.key )  //不能打印 
}
```
